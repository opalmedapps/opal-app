NewsBanner
service in module MUHCApp

Description

API services used to display message banner alerts for the app, e.g. internet connectivity banners, notification banners, etc. For more information on the plugin, Cordova Toast Plugin

Dependencies

  • $cordovaNetwork
  • $filter
  • $translatePartialLoader

Methods

  • setAlertOnline()

    Displays no internet alert

  • showAlert(type)

    Shows alert for the three types, no internet, online, and notifications

    Parameters
    ParamTypeDetails
    typeString

    Type for alert.

  • showCustomBanner(message, color, callback, duration)

    Displays alert based on the parameters

    Parameters
    ParamTypeDetails
    messageString

    Message for the alert

    colorString

    Color for the alert

    callbackFunction

    Callback function for the alert

    durationNumber

    Duration in milliseconds

  • showNotificationAlert(numberOfNotifications, callback)

    Displays the notification banner for the numberOfNotifications specified.

    Parameters
    ParamTypeDetails
    numberOfNotificationsString

    Number of notifications in alert

    callbackFunction

    Callback function

Properties

  • alertTypes

    Alert mappings for types, Returns an object with message, duration, colorf for the specific types. The types are the following: nointernet, connected, notifications
