DeviceIdentifiers
service in module MUHCApp

Description

Service that deals with the device identifiers, sends the identifiers to backend to be used by the push notifications system.

Dependencies

  • RequestToServer

Methods

  • getDeviceIdentifiers()

    Returns
    Object

    Returns deviceIdentifiers object.

  • sendIdentifiersToServer()

    If the device identifiers are set and have not been sent, it sends the device identifiers.

  • setDeviceIdentifiers()

    Sets the deviceIdentifiers property.

  • setIdentifier(identifierType, value)

    Sets the identifierType property for the deviceIdentifiers object

    Parameters
    ParamTypeDetails
    identifierTypeString

    Name of one of the three properties for the deviceIdentifiers object.

    valueString

    new value for field.

  • setSendStatus()

    Sets the haveBeenSent flag

Properties

  • deviceIdentifiers

    Object contains three properties registrationId, deviceUUID, and deviceType, the object is sent to the server to update the devices for a particular user.

  • haveBeenSent

    Flag to check whether the device identifiers have been sent
