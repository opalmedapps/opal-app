RequestToServer

Description

  • sendRequestWithResponse(typeOfRequest, parameters, encryptionKey, referenceField, responseField)

    Sends request to server, awaits for response, and returns with the results from server.

    Parameters
    ParamTypeDetails
    typeOfRequestString

    Type of request

    parametersObject

    Object to be sent to backend as request parameter

    encryptionKeyString

    If defined, its used as a encryption key for data, if not defined, the password hashed is used instead.

    referenceFieldString

    The reference to the request firebase field

    responseFieldString

    The reference to the response firebase field

    Returns
    Promise

    If data returns and the data was processed with a code 1 then it resolves to the data, otherwise it rejects with the error given by the request response
