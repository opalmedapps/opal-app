PlanningSteps
service in module MUHCApp

Description

Sets and determines the current planning step for the user. The planning sequence is defined as follows:

  1. Patient gets a CT Scan
  2. Physician prepares the plan
  3. The radiation dose is calculated
  4. The plan is QAed by physics
  5. The patient is ready for treatment

  • Appointments
