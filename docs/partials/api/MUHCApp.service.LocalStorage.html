LocalStorage
service in module MUHCApp

Description

API to interact with the saved data storage for the patient.

Dependencies

  • $firebaseAuth
  • UserAuthorizationInfo
  • EncryptionService

Methods

  • ReadLocalStorage(section)

    Reads section from localStorage, decrypts and returns decrypted object by using the hashed of password.

    Parameters
    ParamTypeDetails
    sectionString

    Section for patient data in localStorage

    Returns
    Object

    Returns decrypted object representing that section in localStorage

  • WriteToLocalStorage(section, data)

    Writes the data parameter to that section in the patient storage encrypting all the data saved in the process.

    Parameters
    ParamTypeDetails
    sectionString

    Patient section to write to

    dataObject

    Object to write to that particular patient section. i.e. 'Documents', 'Announcements'

  • isUserDataDefined()

    Returns
    Boolean

    Returns value depending on whether the storage for the user exists.

  • resetUserLocalStorage()

    Deletes all the user fields in local storage

  • updateLocalStorageAfterPasswordChange()

    Updates the whole storage by encrypting all information with the new password.
