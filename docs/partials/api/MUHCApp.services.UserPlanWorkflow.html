UserPlanWorkflow
service in module MUHCApp

Description

Organizes task and appointments for the plan workflow and sets properties ready for usage in the Treatment Plan view.

Dependencies

  • $filter

Methods

  • getCurrentTaskOrAppointment()

    Returns
    Array

    Returns array with the tasks and appointments for the plan workflow organized chronologically

  • getTimeBetweenEvents()

    Returns
    Object

    Returns the timeDiff object property properly initialized.

  • setUserPlanWorkflow(tasksAndAppointments)

    Obtains plan workflow object from Firebase through the UpdateUI service. Defines the TasksAndAppointmentsArray by organizing the stages chronologically. Sets the current stage by finding the min time between today and the available stages, and setting CurrentTaskOrAppointmentIndex.

    Parameters
    ParamTypeDetails
    tasksAndAppointmentsObject

    Object contains user's plan workflow.

  • setUserPlanWorkflow()

    Returns
    Array

    Returns the TasksAndAppointmentsArray property.

Properties

  • CurrentTaskOrAppointmentIndex

    Contains index of current stage in the TasksAndAppointmentsArray

  • TasksAndAppointmentsArray

    Contains stages of the plan workflow organized choronologically

  • timeDiff

    An array that contains the time difference between two events, structure dateDiff[i]={Stages: nameStage[i+1]-nameStage[i], timeDiffInDays:date}
