Notifications
service in module MUHCApp

Description

API service used to patient notifications. This Service is deeply linked to other services to extract that information about the actual content of the notification.

Dependencies

  • UserPreferences
  • RequestToServer
  • LocalStorage

  • Appointments

  • Documents
  • EducationalMaterial

Properties

  • Notifications

    Initializing array that represents all the information for Notifications, this array is passed to appropiate controllers.

  • notificationTypes

    Array contains all the mappings to search the actual post for the notification, the icon, the color and the Name of the field. 

         //Example of the Document notification type.
     var notificationTypes={
      'Document':
      {
        icon:'fa fa-folder',
        color:'darkorange',
        NameEN:'AliasName_EN',
        NameFR:'AliasName_FR',
        SerNum:'DocumentSerNum',
        readFunction:Documents.readDocument,
        searchFunction:Documents.getDocumentBySerNum,
        namesFunction:Documents.getDocumentNames,
        PageUrl:Documents.getDocumentUrl
      }...

  • notificationsLocalStorage

    Initializing array that represents all the notifications as saved in localStorage.
