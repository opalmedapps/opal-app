A multiplatform application built using Apache Cordova. The project was written using AngularJS framework , and OnsenUI Framework. This project aims to aid patients in radiation oncology at the Glen Hospital in Montreal, Quebec. The app features: Ability for patients to obtain their documents, such as their images, and test results. Look up of their Appointment calendar and details with a checkin for appointment capability, patient message system with their doctors, a treatment plan overview and progress, educational material about the hospital and specific to their illness, maps of the hospitals. Additionally the patient can make changes to their personal account, and request changes to appointments. They will also be able to add their upcoming appointments to their native device calendar. The main module for the project: MUHCApp. The external dependecies for the project: ProjectDependencies.
