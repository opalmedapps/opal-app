Welcome to Qplus Hybrid Application Admin Portal !
Use the navigation bar to access different administrating panels or see help section for more details. You need valid admin credentials to use any of the tabs.

This is a portal for Q+ Hybrid Application's administrating staff to perform different tasks. The services accessible are :

  • Patient Registration : You can Register new patients for Qplus App or an Alias using their Medicare number, only If their record is present in the ARIA database.
  • Patient Manager : Look up already registered patients and see their information and also Remove existing user accounts.
  • Requests: Browse appointment change requests submitted by patients and remove them.
  • Messages : Send files and messages to the patients through their mobile app.
  • Feedback: Browse feedback messages submitted by the users.
  • Admins: Create,Remove and See the administrators accounts for this panel.
